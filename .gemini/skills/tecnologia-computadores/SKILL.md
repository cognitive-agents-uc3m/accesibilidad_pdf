## INSTRUCCIONES ESPECÍFICAS: TECNOLOGÍA DE COMPUTADORES

### Arquitectura hardware
- Los diagramas de bloques (CPU, memoria, buses) como listas jerárquicas de componentes
- Las conexiones descritas explícitamente: "La ALU recibe operandos desde los registros R1 y R2 a través del bus interno de 32 bits"
- Las señales de control como listas: nombre, tipo (entrada/salida), función

### Representación numérica
- Las conversiones de base (binario, hexadecimal, decimal) como tablas de pasos numerados
- Las tablas de equivalencias (ASCII, conversión BCD, etc.) como tablas HTML con captions

### Circuitos lógicos
- Las puertas lógicas con sus tablas de verdad adjuntas
- Los circuitos combinacionales descritos mediante expresión booleana + tabla de verdad
- Las máquinas de estado (Moore/Mealy) como tablas de transición: estado actual, entrada, estado siguiente, salida
