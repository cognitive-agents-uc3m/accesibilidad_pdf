# Accesibilidad PDF — UC3M

Herramienta web para convertir material docente universitario en formato PDF a **HTML accesible**, optimizado para su uso con lectores de pantalla y otras tecnologías de apoyo.

**Acceso directo:** [accesibilidad-pdf-66041181251.europe-southwest1.run.app](https://accesibilidad-pdf-66041181251.europe-southwest1.run.app)

---

## ¿Qué hace?

El profesorado sube un PDF de cualquier asignatura y la herramienta genera automáticamente un documento HTML que:

- Es completamente navegable con lector de pantalla
- Mantiene la jerarquía de títulos, listas, tablas y figuras con semántica correcta
- Incluye descripciones textuales de imágenes, diagramas y fórmulas
- Cumple las pautas de accesibilidad web WCAG 2.1

El resultado se puede **previsualizar**, **descargar** como fichero HTML o **copiar** directamente.

---

## Funcionalidades principales

### Conversión de PDF a HTML accesible
Selecciona la asignatura, sube el PDF y el modelo de IA genera el HTML adaptado a las instrucciones específicas de cada materia (fórmulas matemáticas, capturas de pantalla de interfaces, código fuente, etc.).

### Instrucciones activas por asignatura
Cada asignatura tiene un conjunto de instrucciones específicas que guían la conversión. Se pueden activar o desactivar individualmente, y añadir instrucciones personalizadas que se guardan entre sesiones.

### Verificación de compatibilidad
Comprueba que las instrucciones activas no se contradigan entre sí antes de lanzar la conversión.

### Auditoría WCAG
Una vez generado el HTML, se puede analizar automáticamente para detectar posibles violaciones de accesibilidad según las pautas WCAG.

### Contenido Accesible
Pestaña con el historial de todos los documentos generados, organizados por asignatura. Permite visualizar, descargar o eliminar cualquier conversión anterior.

---

## Acceso

La aplicación requiere inicio de sesión con cuenta institucional UC3M. El acceso está restringido al personal autorizado.
